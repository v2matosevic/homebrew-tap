# Version2 Homebrew Tap

Homebrew casks for [Version2](https://github.com/v2matosevic) apps.

## Usage

```bash
brew install --cask v2matosevic/tap/winhub
```

Or tap first, then install by name:

```bash
brew tap v2matosevic/tap
brew install --cask winhub
```

## Available casks

| Cask | Description |
| --- | --- |
| [`winhub`](Casks/winhub.rb) | [WinHub](https://github.com/v2matosevic/WinHub) — menu-bar hub of small macOS tweaks for people who switched from Windows. |
